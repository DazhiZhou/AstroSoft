# AstroSoft

This git repo will install related Astronomical Software automatically.

*Pulsar installation*
If you just want to install pulsar suite packages, just type in *python installPulsar.py*
It will install the following packages:
- pgplot
- tempo
- tempo2
- sigproc
- presto
- psrcat
- cfitsio
- calceph
- psrchive
- dspsr
